# idean3885.github.io

기술, 방법론, 그리고 성장의 기록

## 소개

개발자로서의 기술적 탐구, 방법론의 진화,
운영 노하우와 의사결정의 기록을 담는 블로그입니다.
프로젝트 경험과 도구 적응 과정에서 얻은 실전 지식을 공유합니다.

## 카테고리

| 카테고리 | 용도 |
|---|---|
| 개발 기록 | 프로젝트 경험기, 도구 적응기 |
| 생각과 방법론 | 사상, 철학, 설계 원칙, 의사결정 |
| 기술 노하우 | 운영 경험 기반 실전 지식 |
| 아티클 정리 | 외부 글 요약/리포스팅 |
