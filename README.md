# blog.idean.me

기술, 방법론, 그리고 성장의 기록

블로그: <https://blog.idean.me>

## 기술 스택

[Jekyll](https://jekyllrb.com/) + [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) 테마,
[GitHub Pages](https://pages.github.com/)로 배포.

## 개발 환경

이 레포는 콘텐츠(`_posts/`)만 관리하며, 개발 도구는 글로벌 플러그인으로 분리되어 있습니다.

| 플러그인 | 역할 |
|----------|------|
| [claude-devex](https://github.com/idean3885/claude-devex) | 개발 워크플로우 — 이슈·커밋·PR 플로우 자동화 |
| [claude-content-publisher](https://github.com/idean3885/claude-content-publisher) | 콘텐츠 발행 — 원고를 블로그 포스트로 변환·발행 |
